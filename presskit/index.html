Description

Slapped Silly Studios is a three-person student game dev team featuring programmer/artist Rajin Shankar, systems/level designer Dakota Erickson, and designer/producer Ryan Reske. The trio's first project, Slug Slasher, is poised for a summer 2018 release.

History

From Rags to Slightly Fancier Rags

Slapped Silly Games started life as a project with Indiana University's Hoosier Games club. The three-person team then continued work on Slug Slasher over the summer, showcasing the game at Indy Pop Con. With the end in sight, the group is currently preparing for a mid 2018 PC release.
